# LIPU DESIGN REFERENCES

## Vision

We are NOT building a UPVC website.

We are building a luxury architectural experience.

The brand should feel like a hybrid of:

* Apple
* Pella
* Marvin
* ArchDaily
* Luxury architecture studios

The website should make visitors feel:

* Luxury
* Craftsmanship
* Transformation
* Trust
* Aspirational lifestyle
