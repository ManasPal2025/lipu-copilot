## Journey Map: First-Time Visitor

### Scenario
Priya (homeowner) discovers the website via Google search for "UPVC windows Mumbai"

### Steps

**1. Landing (0-30 seconds)**
```
User Action:
в”ңв”Җв”Җ Lands on homepage from Google search
в”ңв”Җв”Җ Views hero section with product showcase
в”ңв”Җв”Җ Observes luxury, modern design
в””в”Җв”Җ Sees clear navigation

Touchpoint: Homepage
Design Goals:
в”ңв”Җв”Җ Professional, premium first impression
в”ңв”Җв”Җ Clear value proposition
в”ңв”Җв”Җ Easy navigation visible
в””в”Җв”Җ Trust signals (awards, certifications)

Success Metric: User scrolls below fold
```

**2. Exploration (1-5 minutes)**
```
User Action:

в”ңв”Җв”Җ Reads product categories
в”ңв”Җв”Җ Views featured products

**3. Product Discovery (5-15 minutes)**
```
User Action:

в”ңв”Җв”Җ Sees product listings with images
в”ңв”Җв”Җ Reads product descriptions

в”ңв”Җв”Җ Time spent on product
в”ңв”Җв”Җ Filters applied
в””в”Җв”Җ Search queries

"I see you're interested in sliding windows. 
Would you like to know about the best glass 
type for coastal areas, or shall I show you 
color options?"
```

**4. AI Consultation (Optional - 5-10 minutes)**
```
User Action:
в”ңв”Җв”Җ Clicks "Chat with AI Assistant"
в”ңв”Җв”Җ Asks question: "What's best for coastal weather?"
в”ңв”Җв”Җ Receives answer with product recommendations

в”ңв”Җв”Җ Product recommendations
в”ңв”Җв”Җ Quote preview
в””в”Җв”Җ "Book Site Visit" button

в”ңв”Җв”Җ Questions asked
в”ңв”Җв”Җ Products recommended
в”ңв”Җв”Җ Average response satisfaction
в””в”Җв”Җ Lead quality score
```

**5. Quote Request (10-20 minutes)**
```
User Action:
в”ңв”Җв”Җ Clicks "Request Quote"

в”Ӯ   в”ңв”Җв”Җ Project location
в”Ӯ   в”ңв”Җв”Җ Window/Door count
в”Ӯ   в”ңв”Җв”Җ Preferred style
в”Ӯ   в””в”Җв”Җ Budget range
в”ңв”Җв”Җ Selects products
в”ңв”Җв”Җ Provides contact info
в””в”Җв”Җ Submits

Touchpoint: Quote Request Form
в”ңв”Җв”Җ Multi-step form (UX progressive disclosure)
в”ңв”Җв”Җ Pre-filled with browsed products
в”ңв”Җв”Җ Location auto-complete
в”ңв”Җв”Җ Image upload option
в””в”Җв”Җ Email/SMS consent

Confirmation:
в”ңв”Җв”Җ Email with quote number
в”ңв”Җв”Җ SMS notification
в”ңв”Җв”Җ "Track Your Quote" link
в”ңв”Җв”Җ Next steps communication

в”ңв”Җв”Җ Lead score calculated
в”ңв”Җв”Җ Follow-up scheduled
в””в”Җв”Җ Notification to sales team
